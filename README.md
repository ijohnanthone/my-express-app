# My Express App 🚀

This is a simple Node.js and Express application created as part of a class activity to understand how Express works.

## 📌 Features

- Express server setup
- Static file serving
- Basic routing

## 🛠️ Technologies Used

- Node.js
- Express.js

## ⚙️ Installation

Clone the repository:

git clone https://github.com/ijohnanthone/my-express-app.git

Go into the project folder:

cd my-express-app

Install dependencies:

npm install

## ▶️ Run the Application

node index.js

Open your browser and visit:

http://localhost:3000
